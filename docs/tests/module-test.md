# Модульные тесты - Backend

:::info
Проверяет [Backend](/definitions#backend) сущности на корректность работы, очень быстрое. Покрытие на данном этапе маленькое, команда стремится его увеличить.
:::

## Запуск

Запустить модульное тестирование можно командой:
```bash
pytest app -v
```

## Проверяемые сущности:

1. `JwtAuthService`
1. `TgBotAuthService`
1. `AuthorizationService`
