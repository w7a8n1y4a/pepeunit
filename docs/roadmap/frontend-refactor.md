# Рефатор фронтенда

## Актуализация стиля

Переход на shadcn, скорее всего придётся написать где-то кастомные решения, на основе компанентов из shadcn

## Перевызовы api

Найти лишние повторные вызовы, возникающие из-за useEffect, на версии 0.9.0 аналогично были убраны лишине падающие запросы. Можно внедрить на бек временные debug коментарии, которые бы показывали больше информации о приходящих туда запросах

## Ререндеры

Расствить каунтеры и проверить частоту ререндеров. Вычислить время этих ререндеров, если быстрые то пофиг, если медленные фиксить
