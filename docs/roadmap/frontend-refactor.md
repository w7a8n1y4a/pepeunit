# Рефатор фронтенда

## Актуализация стиля

Переход на shadcn, скорее всего придётся написать где-то кастомные решения, на основе компонентов из shadcn

## Перевызовы api

Найти лишние повторные вызовы, возникающие из-за useEffect, на версии 0.9.0 аналогично были убраны лишние падающие запросы. Можно внедрить на бек временные debug комментарии, которые бы показывали больше информации о приходящих туда запросах

## Ререндеры

Расствить каунтеры и проверить частоту ререндеров. Вычислить время этих ререндеров, если быстрые то без разницы, если медленные - фиксить.
