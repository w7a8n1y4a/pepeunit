# Поддержка проекта

:::info
Я разрабатываю этот проект в одиночку. Любая ваша поддержка мотивирует меня продолжать развивать проект, исправлять баги и добавлять новые функции.

Если вам нравится [Pepeunit](/conception/overview), вы можете поддержать проект финансово, чтобы ускорить разработку и повысить качество продукта:
:::

`Ethereum (ETH)`:

```

```

`Bitcoin (BTC)`:

```
bc1q537kletr80tg9s3cc47920cd5qcef0dwly08qr
```

:::warning
Монеты будут расходоваться на улучшение проекта:
1. Привлечение профильных разработчиков для повышения качества кодовой базы или разработку новых фич
2. Оплату дополнительной инфраструктуры, если такая потребуется
3. Оплату работы дополнительных специалистов в смежных областях: аналитиков, юридических консультантов, видео-монтажёров и др.
4. Закупку [IoT](/definitions#iot) компонентов для тестирования библиотек и/или компонентов [Pepeunit](/conception/overview). Закупленные элементы не будут перепроданы без решения сообщества, но могут быть переданы разработчикам [Pepeunit](/conception/overview) для тестов.

В случае расходования значительных средств будет объявлено голосование в https://t.me/pepeunit, по результатам которого средства будут направлены на наиболее привлекательный для сообщества вариант.

В голосовании всегда будет вариант "Никакой из предложенных", чтобы у сообщества была возможность отменить голосование и предложить другие варианты
:::

:::danger
Все монеты будут потрачены исключительно на [Pepeunit](/conception/overview)

Все расходы средств, собранных через пожертвования, будут отображаться в `CSV` файле - [Детализация Расходов Проекта](https://minio.pepemoss.com/public-data/csv/detail.csv)
:::
