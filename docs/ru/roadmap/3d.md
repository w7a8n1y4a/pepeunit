## Фреймовые корпуса

Жёсткий корпус который идёт вокруг платы, при этом имеет отверстия для проводов

Основная идея, чтобы у них были поверхности для укладки стёкол и/или заливки этих пространств в эпоксидку/селикон. Такие корпуса будут влаго/пылестойкие, антивибрационные или ударостойкие

Можно делать полуфрейм, узогнутый фрейм или какие-то другие фигуры

Для закрепления плат, нужно придумать зажим с гужёном, но при этом он должен быть пинцетного типа, желательно с падаллельным сдвиганием губок

Можно делать в фреймах отверстия для usb c и любые другие элементы дизайн системы

Для разьёмов делать отверстия с учётом подключения
