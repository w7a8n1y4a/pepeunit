# Clickhouse

:::warning
Ручное заполнение обычно не требуется, нужно заполнить только основные переменные т.к. есть [файл генерации .env файлов для сервисов](/deployment/docker/deploy#заполнение-первичного-env-фаила)
:::

:::warning
Полный список переменных окружения доступен в [документации Clickhouse](https://clickhouse.com/docs)
:::

Переменная | Пример | Зачем нужна?
-- | -- | --
`CLICKHOUSE_USER` | `admin` | Имя пользователя базы данных
`CLICKHOUSE_PASSWORD` | `mypassword` | Пароль пользователя базы данных
`CLICKHOUSE_DB` | `default` | Название базы данных
