# Clickhouse

[Clickhouse](https://clickhouse.com/) - колончатая аналитическая СУБД с открытым исходным кодом, позволяющая выполнять аналитические запросы на структурированных данных при помощи языка [SQL](/definitions#sql).

:::info
[Backend](/definitions#backend) хранит в ней слудующую информацию:
1. Логи [Unit](/definitions#unit)
1. Данные [DataPipe](/definitions#datapipe)
:::
