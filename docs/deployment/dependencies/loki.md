# Loki

`Loki` — это система агрегации и хранения логов с открытым исходным кодом, разработанная `Grafana Labs`, предназначенная для сбора, индексации и анализа логов в распределённых системах.

:::info
`Loki` в [Pepeunit](/conception/overview) используется для агрегации данных из [Promtail](/deployment/dependencies/promtail) и предоставления этих данных для [Grafana](/deployment/dependencies/grafana)
:::
