# Loki

`Loki` is an open-source log aggregation and storage system developed by `Grafana Labs`, designed for collecting, indexing and analyzing logs in distributed systems.

::::info
In [Pepeunit](/en/conception/overview), `Loki` is used to aggregate data from [Promtail](/en/deployment/dependencies/promtail) and to provide this data to [Grafana](/en/deployment/dependencies/grafana).
::::
