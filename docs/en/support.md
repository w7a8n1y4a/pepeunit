:::warning
Funds will be spent on improving the project:
1. Hiring specialized developers to improve the codebase quality or develop new features
2. Paying for additional infrastructure if needed
3. Paying for additional specialists in related fields: analysts, legal consultants, video editors, etc.
4. Purchasing [IoT](/en/definitions#iot) components for testing libraries and/or [Pepeunit](/en/conception/overview) components. Purchased items will not be resold without a community decision, but may be provided to [Pepeunit](/en/conception/overview) developers for testing.

In the event of significant expenditures, a vote will be announced in https://t.me/pepeunit, based on which funds will be directed to the option most attractive to the community.

There will always be a "None of the above" option in the vote, so the community can cancel the vote and suggest other options
:::

:::danger
All funds will be spent exclusively on [Pepeunit](/en/conception/overview)

All expenses of funds collected through donations will be displayed in a `CSV` file - [Project Expense Details](https://minio.pepemoss.com/public-data/csv/detail.csv)
:::
