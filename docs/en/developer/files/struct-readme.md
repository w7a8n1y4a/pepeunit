# readme.md

[readme.md](/en/definitions#readme-md) is the documentation for a [Unit](/en/definitions#unit) and the most important file in any repository. How easy it is for other people to use a [Unit](/en/definitions#unit) depends directly on how complete and clear this file is.

::::tip
To keep [Unit](/en/definitions#unit) documentation consistent, every [Pepeunit](/en/conception/overview) instance provides a [readme.md](/en/definitions#readme-md) generator based on [pepeunit.toml](/en/definitions#pepeunit-toml). The generator converts machine-readable [pepeunit.toml](/en/definitions#pepeunit-toml) into a human-readable [readme.md](/en/definitions#readme-md).
::::
