# Unit tests - Backend

::::info
Checks [Backend](/en/deployment/dependencies/backend) entities for correct behavior and runs very fast. Test coverage is currently small, and the team is working to increase it.

## Run

You can run unit tests with:
```bash
pytest app -v
```
